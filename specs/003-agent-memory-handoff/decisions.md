# Decisions

- Conduit compiles context; agents consume context packs and return handoff envelopes.
- Agents never open, query, mutate, or receive either Turso database.
- Feature 003 consumes Feature 002 persistence and does not duplicate database bootstrap or migration execution.
- Memory is curated, evidence-backed, compact, lifecycle-managed knowledge, not transcript archival or generic repository RAG.
- Repository indexing and memory corpus are separate concepts.
- Deterministic and lexical retrieval are required; embeddings and Turso vector retrieval are optional enhancement channels.
- Initially embed only active validated memory records, never full source files or raw transcripts.
- Similarity cannot override eligibility, evidence validity, exact contracts, role/path restrictions, or lifecycle state.
- Global memory promotion is explicit, generalized, provenance-backed, and revocable/supersedable.
